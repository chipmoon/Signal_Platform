# Trading Intelligence Platform 📈

A professional multi-asset trading analysis platform powered by machine learning and institutional-grade strategies.

## Features

- 📡 **Market Pulse** — Real-time multi-asset overview
- 📊 **Dashboard** — Backtest performance & equity curves
- 🧠 **Sentiment Hub** — News NLP + Social + Fear & Greed
- 🔥 **Alpha Scanner** — Multi-strategy signal scanner
- 🤖 **AI Forecast** — ML-driven multi-horizon predictions
- 🌐 **Geo-OSINT Lab** — Geopolitical market intelligence
- 🛡️ **Risk Monitor** — Drawdown, VaR, position sizing
- 📈 **Asset Manager** — Portfolio construction tools

## Data Sources

- Yahoo Finance (yfinance)
- CFTC COT Reports
- Alternative.me Fear & Greed Index
- VNStock (Vietnamese market data)

## Tech Stack

- **Framework**: Streamlit
- **ML**: scikit-learn, LightGBM
- **Data**: pandas, numpy
- **Charts**: Plotly
